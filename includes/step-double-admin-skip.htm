Double Admin Skip

Freeze both admins and power fuse before dying

  • Run to about where the tile texture comes to a point in the middle of the arena.
  • Turn toward and throw an ice bomb at each admin.
  • Activate the fuse.
  • Run back a bit, wait 1-2 seconds, then run toward the bottom and cross the bridge.
  • While waiting, switch out ice-bombs for the orb.
If you aren't in critical HP, immediately after activating the fuse is a good chance to do it.
