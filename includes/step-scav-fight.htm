Boss: Scavenger (Fight)

Fight the Scavenger with ice-combo

  1. Equipment required: Sword, Dagger, Wand. Make sure the speedrunner code is active.
  2. Run up to the Scavenger and execute the ice combo.
  3. Obtain the Blue Key, equip a bomb, and use it to death warp. Be sure to use the death animation to menu.
