Boss: Scavenger (Fight)

Fight the Scavenger with ice-combo

  • Make sure the speedrunner code is active.
  • Equipment required: Sword, Dagger, Wand
  • Run up to the Scavenger and execute the ice combo.
  • Obtain the Blue Key, equip a bomb, and use it to death warp.
  • Be sure to use the death animation to menu.
