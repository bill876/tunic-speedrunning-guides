Boss: The Librarian (Fight)

Fight the librarian with ice-combo

  • Type in the speedrunner code while climbing the ladder.
  • Approach the boss and:
    • If you have Muffling Bell on: 1-2-3 combo, then ice combo.
    • If not, start ice combo as the boss is spinning to face you.
  • Retrieve the key, throw a bomb, and death warp.
